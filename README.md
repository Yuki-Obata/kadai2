## Dependency  
python 
## Usage 
ノードの実行: rosrun mypkg count.py  
twice.pyの実行: rosrun mypkg twice.py (2倍になった数字が表示される)  
実行: rostopic echo /twice (ノードを立ち上げ、トピックとしてデータを得る)
